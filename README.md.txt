# Guess My Number Game

Welcome to the Guess My Number game! This simple game is built using vanilla HTML, CSS, and JavaScript.

## How to Play

1. Open the `index.html` file in your web browser.
2. Enter a guess in the input field and click the "Submit" button.
3. Receive feedback on your guess: whether the correct number is higher or lower.
4. Repeat steps 2 and 3 until you guess the correct number.
5. Once you guess the correct number, the game will display the number of attempts you took to win.

## Files and Structure

- `index.html`: The main HTML file that contains the structure of the game.
- `style.css`: The CSS file that styles the game interface.
- `script.js`: The JavaScript file that handles the game logic.

## Game Logic

The game generates a random number between a specified range. The player needs to guess this number, and the game provides feedback on whether the guess is too high or too low. The game ends when the player correctly guesses the number.
